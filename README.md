# 中国科研常用LaTeX模板集

整理中国科研常用的LaTeX模板，包括国自然科学基金的正文模板、毕业论文等。

## 背景

经常用LaTeX进行论文、标书写作；但网上的资源比较零散，且距离实用有相当距离。这是整理我修改过的模板，希望对大家有帮助！
